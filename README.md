# Test Project

A simple Node.js project demonstrating basic functionality with Jest testing.

## Features

- Hello World function
- Utility functions (add, multiply, capitalize)
- Complete test coverage with Jest

## Installation

```bash
npm install
```

## Usage

Run the app:

```bash
npm start
```

Run tests:

```bash
npm test
```

## Project Structure

```
test-project/
├── src/
│   ├── index.js      # Main entry point
│   └── utils.js      # Utility functions
├── tests/
│   └── index.test.js # Jest tests
├── package.json

```
